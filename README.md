How to Run
Clone the repository: git clone https:(https://github.com/yalda1382/store_shop/)

Create virtual environment and activate it :

python3 -m venv venv

active windows: venv\Scripts\activate

active mac & linux :source venv/bin/activate

Install dependencies: pip install -r requirements.txt

Run server: python manage.py runserver

8.localhost: http://127.0.0.1:8000/
